№ 6 «Иследование работы микросхемы демультиплексора»

 

 

1.      Включить компьютер, открыть папку Электроника, распаковать файл «05623_programma_Elektronik_Workbehch.rar» и запустить программу файлом WEWB32.exe.

2.       Собрать схему (рис.1) для исследования микросхемы дешифратора-демультиплексора КР1533ИД7 (аналог - микросхема SN74АLS138 фирмы «Texas Instruments»).

 

Рис.1

 

Новый элемент, который появляется в этой работе:

 

- микросхема дешифратора-демультиплексора КР1533ИД7 (аналог - микросхема SN74АLS138 фирмы «Texas Instruments»). Его можно найти при нажатии клавиши «Digit» в нижней строке верхнего меню и дальнейшем выборе нужной микросхемы

 

 

 

 

3.       Развернуть панель генератора сигналов и установить в первой строке значения входных сигналов для заданной коммутации в практическом задании.

Например: в практическом задании требуется подключение информационного входа  D (здесь за информационный вход D принят стробирующий вход G1) к выходу Y3, когда на этот вход поступает сигнал 0.  Для этого указанный  сигнал  0 подадим на информационный  вход  D (выход 1 генератора сигналов). На адресных входах АВС (выходы 5,6,7 генератора сигналов) установим трехразрядный код 011, соответствующей цифре 3, так как задан выход Y3. На стробирующем входе G2A и G2D (выходы 9,10 генератора сигналов) установим активный сигнал 0 (эти входы инверсные статические), которые дадут разрешение на срабатывание демультиплексора. Выходы 2,3,4,8,11-16 генератора в данном случае не используются, поэтому здесь можно устанавливать любые сигналы, для примера поставим 0. В результате на выходах генератора сигналов нужно установить: 0000000000110000, что в 16-й системе счисления дает: 0030 (см. рис.2).

 

Рис.2

 

4.       В позиции Final генератора сигналов установить 0000 (рис.2).

5.       Нажать на клавишу Step генератора сигналов. Сравнить полученный результат с данными практического задания, сделать вывод и показать результат преподавателю.

Например, для указанного в пункте 3 примера подается сигнал 0 на вход D. На входы АВС подаются сигналы 011 (адрес выхода Y3), на входы G2A,G2B подаются разрешающие активные сигналы 0. В результате сигнал 0 со входа D проходит на выход Y3 (так как этот выход инверсный, то здесь указанный сигнал 0 преобразуется в 1). На остальных выходах устанавливаются пассивные сигналы 1.
