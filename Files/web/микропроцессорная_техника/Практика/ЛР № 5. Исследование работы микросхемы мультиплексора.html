2.      

Собрать схему (рис.1) для исследования микросхемы мультиплексора КР1533КП7 (аналог - микросхема SN74АLS151 фирмы «Texas Instruments»).

3.       Развернуть панель генератора сигналов и установить в первой строке значения входных сигналов для заданной коммутации в практическом задании.

Например: в практическом задании требуется подключение входа D1 к выходу, когда на заданный вход D1 поступает сигнал 0.  Для этого указанный  сигнал  0 подадим на нужный  информационный  вход  D1 (выход 2 генератора сигналов). На адресных входах АВС (выходы 9,10,11 генератора сигналов) установим трехразрядный код 001, соответствующей цифре 1, так как задан вход D1. На стробирующем входе G (выход 12 генератора сигналов) установим активный сигнал 0, который даст разрешение на срабатывание мультиплексора. Что подавать на входы D0, D2-D7 – безразлично, так как эти входы в данном случае не подключены; поэтому установим для определенности нули. Выходы 13-16 генератора в данном случае не используются, поэтому здесь можно устанавливать любые сигналы, для примера поставим 0. В результате на выходах генератора сигналов нужно установить: 0000000100000000, что в 16-й системе счисления дает: 0100 (см. рис.2).

 

Рис.2

 

4.       В позиции Final генератора сигналов установить 0000 (рис.2).

5.       Нажать на клавишу Step генератора сигналов. Сравнить полученный результат с данными практического задания, сделать вывод и показать результат преподавателю.

Например, для указанного в пункте 3 примера подается сигнал 0 на вход D1. На входы D0,D2-D7 можно подавать любые сигналы, для примера установлены нули. На входы АВС подаются сигналы 001 (адрес информационного входа D1), на вход G подается разрешающий активный сигнал 0. В результате сигнал 0 со входа D1 проходит на выход Q (на инверсном выходе  устанавливается сигнал 1).
