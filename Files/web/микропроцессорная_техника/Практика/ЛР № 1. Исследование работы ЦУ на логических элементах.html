№ 1  «Исследование работы ЦУ на логических элементах»

 

 

1.      Включить компьютер, открыть папку Электроника, распаковать файл «05623_programma_Elektronik_Workbehch.rar» и запустить программу файлом WEWB32.exe.

2.       Собрать схему комбинационного ЦУ своего варианта из практического задания № 1 по микроэлектронике (приведенная на рис.1 схема из конспекта дана для примера).

 

Рис.1

 

Новые приборы и элементы, которые появляются в этой работе:

 

Логические элементы, изображение которых выполнено по «американскому» стандарту; их

можно найти при нажатии клавиши                       в нижней строке верхнего меню.

- индикатор для фиксации сигналов логического нуля и логической единицы (в последнем случае он становится красным); клавиша с красной цифрой 8 в нижней строке верхнего меню.

- генератор сигналов (Word Generator); последняя клавиша в нижней строке верхнего меню.

 

3.       Подключить ко входам схемы Х1,Х2,Х3,Х4 генератор сигналов, а для фиксации сигналов подключить индикаторы ко входам схемы, а также к выходам всех элементов так, как показано на рис.2 (напоминание: эта схема из конспекта дана для примера).

 

Рис.2

 

4.       Развернуть панель генератора сигналов и установить в первой строке заданные в практическом задании по микроэлектронике значения входных сигналов.

Например, если в примере из конспекта заданы входные сигналы 1010 (сигналы всегда читаются снизу вверх!), то в шестнадцатиричном коде это будет цифра А. Тогда в первой строке следует набрать: 000А (как на рис.3).

Рис. 3

 

5.       Нажать на клавишу Step генератора сигналов.

 

6.       Определить по индикаторам значения входных сигналов, а также сигналов на входах и выходах всех элементов. Сравнить эти значения с результатами практического задания по микроэлектронике, сделать вывод и показать преподавателю.
