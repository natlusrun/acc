Практическая работа №1

 

«Анализ работы логических элементов»

 

Цель работы:

 

1.     Закрепление знаний по логическим элементам: названия, выполняемые операции, условные графические изображения, микросхемы логических элементов.

2.     Закрепление практических навыков анализа работы логических элементов.

 

Литература:

Нсанов М.А. Цифровые устройства и микропроцессорные системы – Алматы, 2004, §§ 1.1, 1.2.

 

Подготовка к работе:

1.     Проработать темы "Общие сведения о цифровых сигналах и цифровых устройствах", "Общие сведения о логических элементах", в результате чего знать ответы на контрольные вопросы.

2.     Выполнить домашнее задание.

3.     Подготовить бланк отчета.

 

Исходные данные:

 Каждому учащемуся выдается индивидуальная карточка со схемой ЦУ, построенная на логических элементах.

 

Задание:

1.     Указать название каждого элемента и какую операцию он выполняет.

2.     Выполнить анализ работы схемы в статическом режиме по заданным значениям входных сигналов: указать значения сигналов на входах и выходах всех элементов.

Содержание отчета:

1.     Заданная схема ЦУ, выполненная карандашом, по линейке и с соблюдением всех стандартов.

2.     Перечисление названий всех элементов схемы с указанием выполняемых ими функций.

 

 

Контрольные вопросы:

1.     Что  представляют собой сигналы логического 0 и логической 1 в цифровых устройствах?

2.     Название, выполняемые операции, условное графическое изображение элементов НЕ, И, ИЛИ, И-НЕ, ИЛИ-НЕ.

3.     Уметь указать значение сигнала на выходе любого из перечисленных  элементов при заданных значениях сигналов на его входах.

4.     Микросхемы логических элементов.

 

Вариант № 1

 

 

 

1.     Указать название каждого элемента и какую операцию он выполняет.

 

2.     Выполнить анализ работы схемы в статическом режиме по заданным значениям входных сигналов: указать значения сигналов на входах и выходах всех элементов.

 

3.     По заданной функции построить схему, проставить позиционное обозначение элементов, составить перечень элементов к этой схеме, произвести анализ работы устройства при заданных значениях входных сигналов.
