Формат сообщений UDP

  • Единица данных протокола UDP называется 

UDP- пакетом или пользовательской дейтаграммой.

UDP-пакет состоит из заголовка и поля данных, в котором размещается пакет прикладного уровня. Заголовок имеет простой формат и состоит из четырех двухбайтовых полей:

  • Поле source port - номер порта процесса-отправителя.

  • Поле destination port - номер порта процесса-получателя.

  • Поле message length - длина UDP-пакета в байтах.

  • Поле checksum - контрольная сумма UDP-пакета.
