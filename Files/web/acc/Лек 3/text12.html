Транспортный уровень

  • Основной функцией этого уровня  является доставка сообщений (транспортных блоков), которые состоят из сетевых пакетов.

  • С этой целью транспортные объекты сетевого программного обеспечения организуют разборку сообщений на передающем конце и сборку сообщений из принимаемых пакетов на приемной стороне.

  • Кроме того, транспортный уровень занимается согласованием различных сетевых уровней с помощью соответствующих шлюзов (согласование сетевых объектов принципиально различных сетей) и мостов (согласование сетевых объектов однотипных сетей).
