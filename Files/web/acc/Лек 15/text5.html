Основа работы сети

  • Основу работы сети составляют так называемые сетевые службы (или сервисы).

  • Базовый набор сетевых служб любой корпоративной сети состоит из следующих служб:

  • службы сетевой инфраструктуры DNS, DHCP, WINS;

  • службы файлов и печати;

  • службы каталогов (например, Novell NDS, MS Active Directory);

  • службы обмена сообщениями;

  • службы доступа к базам данных.
